## Quick start

1. Download and unzip anywhere
2. Run `Grey GodZilla DayZ Mod Config Checker v1.19.1.exe`
3. **Config Checker** tab — drop a `config.cpp`, `@Mod` folder, mission path, or server folder
4. Press **Scan** (F5)

### Live RPT watch (v1.19)
1. Set your server **profiles** folder in the sidebar (or browse `…`)
2. Toggle **Watch folder**
3. New RPT errors and mod failures show up automatically while the server runs
